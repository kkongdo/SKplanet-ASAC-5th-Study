iterator(반복문)와 condition(조건문)
 
 * iterator(반복문)
 
 if문은 if뒤에 오는 소괄호 안의 조건식이 참으로 평가되면 중괄호 안의 코드를 실행하는 조건문을 의미한다.
 숫자, 문자열, undefined, null을 써도 무조건 참/거짓으로 평가되며 평가된 참 혹은 거짓에 따라 중괄호 안의 코드를 실행하거나 실행하지 않는다.
 
 ```javascript
 
 if(조건식){
	//조건식이 참이면 실행
 }
 
 //Example
 let num = 8;
 if(num % 2 === 0){
	console.log("변수 num에 할당된 숫자는 짝수");
 }

 ```
 
 P.S) 블록문이란?
 
 블록문(block statement)은 한 개 이상의 자바스크립트 코드를 중괄호로 묶은 것을 의미한다.
 '블록' 혹은 '코드 블록'라고도 한다.
 if문을 설명할 때 '조건식이 참이면 중괄호 안 코드를 실행한다'고 했지만, 이를 다르게 말하면 '조건식이 참이면 블록문을 실행한다'라고 말할 수 있다.
 
 
 else 문
  - if문의 조건식이 거짓일 때 실행되는 블록문을 정의한다.
 ```javascript
	
	if(조건식){
		// 조건식이 참이면 블록문을 실행
	}
	else{
		// 조건식이 거짓이면 블록문을 실행
	}
	
	//Example
	let num = 5;
	if(num % 2 === 0){
		console.log("변수 num에 할당된 숫자는 짝수");
	}
	else{
		console.log("변수 num에 할당된 숫자는 홀수입니다.");
	}
	
 ```
 
 else if문
 
 else if문은 if문에 조건을 추가하고 시픙ㄹ 때 사용한다.
